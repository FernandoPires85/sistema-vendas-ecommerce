# 🛒 Motor de Vendas - E-commerce Simples

Projeto em JavaScript puro que simula um sistema básico de e-commerce.

## 📌 Funcionalidades

- Listagem de produtos
- Verificação de estoque
- Aplicação de desconto de 10%
- Exibição dinâmica no HTML

## 💻 Tecnologias

- HTML
- CSS
- JavaScript

## 🚀 Como executar

1. Baixe o projeto
2. Abra o arquivo `index.html` no navegador
3. Veja o resultado na tela
